## Project Structure

The project is a React-based booking import application with:
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: SQL Server

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure `.env` file with database credentials

3. Start the backend server:
   ```bash
   npm run server
   ```

4. Start the frontend (in a separate terminal):
   ```bash
   npm run dev
   ```
